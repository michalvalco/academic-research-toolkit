# GitHub Pages

This directory can be used for GitHub Pages if you choose to serve the landing page from a `/docs` directory instead of the root.

## Setup Instructions

To enable GitHub Pages for this repository:

1. Go to your repository settings
2. Navigate to **Pages** in the left sidebar
3. Under **Source**, select:
   - **Deploy from a branch**
   - Branch: `main` (or your default branch)
   - Folder: `/ (root)` or `/docs`
4. Click **Save**

Your landing page will be available at:
`https://michalvalco.github.io/academic-research-toolkit/`

## Custom Domain (Optional)

If you have a custom domain, you can:

1. Create a `CNAME` file in the root directory with your domain
2. Configure your DNS provider to point to GitHub Pages
3. Enable HTTPS in repository settings

For more information, see: [GitHub Pages Documentation](https://docs.github.com/en/pages)
