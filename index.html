Get Started in Minutes

Simple installation and intuitive command-line interface.

1

Clone Repository

Download the toolkit from GitHub

git clone https://github.com/michalvalco/academic-research-toolkit.git
2

Install Dependencies

Set up Python environment

pip install pdfplumber pypdf anthropic fastmcp
3

Process PDFs

Start extracting insights

python pdf_processor.py --input ./pdfs --output ./results

Example: Extract Citations

# Extract citations from processed papers python citation_extractor.py --input ./results --output ./citations # Analyze themes across corpus python theme_analyzer.py --input ./results --output ./themes # Generate APA bibliography python bibliography_generator.py --input ./citations/paper.json --output ./bibliography.txt --format apa
