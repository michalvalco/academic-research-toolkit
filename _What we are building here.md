The Tactical Plan for Our Sessions

Each session would follow this pattern:



Concept introduction (10 min) - I explain the core idea conversationally

AI enhancement (10-15 min) - We figure out how to use Claude/ChatGPT/APIs to supercharge what you just built

Between sessions, you experiment. You come back with questions. We iterate.

Tools I'd Actually Recommend

The course mentions Airtable, Glide, Adalo, Zapier. All solid. But let me add some nuance:

Definitely use:



Airtable - phenomenal for database layer, integrates with everything

Make.com - more powerful than Zapier, better for complex workflows, has generous free tier



Consider instead of/alongside:



Softr or Glide - both excellent, Softr might fit your needs better for course delivery

Bubble instead of Adalo if you want more power (steeper learning curve though)

Notion databases + integrations - you might already be using Notion, could be your starting point



Definitely leverage:



Claude API (obviously) for content generation within your apps

Anthropic's prompt caching for cost-effective automation

Your existing AI toolkit to generate content, process data, create variations
